# Lyric SRT Studio

曲を再生しながら歌詞行の開始・終了時刻を記録し、日本語・English・日英併記の SRT 字幕を作成するローカルファーストのブラウザアプリです。音源・歌詞・プロジェクトデータは外部へアップロードされません。

## 主な機能

- 歌詞をまとめて貼り付け、行単位の字幕へ変換
- ファイル選択またはドラッグ＆ドロップで音源を読み込み
- Space キーまたは画面固定ボタンで開始時刻を連続記録
- J / K / L キーで3秒戻る・再生/停止・3秒進む、E キーで終了時刻を記録
- 次の字幕直前まで終了時刻を自動設定
- 必要な行だけ終了時刻を手動指定
- ±0.1秒 / ±0.5秒の微調整、最大100件の Undo / Redo
- スマートフォンでは文字付きの上下ナビと Undo / Redo を画面下部へ固定
- 端末内での波形解析と字幕プレビュー
- 波形と同じ時間軸で確認できる字幕タイムライン
- 曲全体を収める全体表示と、横スクロールできる編集表示
- 記録中はページを動かさず、タイムライン内だけを再生ヘッドへ追従
- 記録漏れ、時刻逆転、長すぎる表示、言語抜けなどの品質チェック
- 日本語・English・日英併記 SRT の書き出し
- UTF-8 BOM の有無を選択可能
- ブラウザへの自動保存
- JSON プロジェクトファイルの保存・読み込み
- 歌詞を残したまま全時刻を一括消去（直後の取り消しに対応）
- 旧バージョンのブラウザ保存データからの移行

## 使い方

1. 「曲を選ぶ」またはドラッグ＆ドロップで端末内の音源を読み込みます。
2. 日本語歌詞と、必要なら英語歌詞を1行ずつ貼り付けて「歌詞一覧に反映」を押します。
3. 曲を再生し、歌い始めに合わせて「開始を記録」または Space キーを押します。記録した歌詞は波形下の字幕タイムラインへ即時表示されます。
4. 長い間奏などで字幕を先に消したい場合は、その行の「終了を記録」を使用します。
5. プレビューと完成度チェックを確認し、必要な言語の SRT を書き出します。
6. 作業を別の端末やブラウザへ移す場合は「プロジェクトを保存」を使用します。音源そのものはプロジェクトに含まれないため、読み込み後に選び直してください。

## 開発チェック

Node.js 18 以降で次を実行します。

```sh
node --test srt-core.test.js
```

`index.html` をブラウザで直接開くだけでも動作します。確認用サーバーやインストールは必要ありません。
