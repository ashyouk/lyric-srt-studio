# Lyric SRT Studio — MVP

ローカルの音声・動画ファイルを再生しながら、歌詞の行ごとに開始時刻を記録して SRT 字幕を作る、依存関係なしのブラウザアプリです。`index.html` をそのまま開いて動くよう、外部モジュールの読み込みも使っていません。

## 使い方

1. `index.html` を Safari / Chrome / Edge で開く。
2. 日本語歌詞をまとめて貼り付け、「歌詞を反映して一覧を作る」を押す。英語歌詞も貼り付ければ、同じ行順で対応する。
3. 端末内の曲ファイルを選び、再生する。iPhoneでは選択画面の形式しぼりを使わず、曲ファイルを選べるようにしている。
4. 曲に合わせて固定操作盤の「この行を記録して次へ」を押す（歌詞行、Space / Enter でも可）。記録後は次の行へ自動で進み、画面も追従する。
5. 日本語・English・日英併記のいずれかを選び、SRTを書き出す。

## 仕様

- 曲は外部へアップロードされず、ブラウザ内で再生される。
- 曲を端末内で解析して波形を表示し、波形を押した位置へ移動できる。解析できない形式でも再生・記録は利用可能。
- 記録した歌詞を曲の時間軸上へ字幕ブロックとして表示。再生ヘッドで現在位置を確認でき、ブロックから該当行へ移動できる。
- 0.5 / 0.75 / 1.0 倍速、3秒戻し、直前操作の取消に対応。
- 記録後の時刻を ±0.1秒 / ±0.5秒単位で微調整できる。
- 歌詞の下書きと記録時刻は、同じブラウザの localStorage に保存される。
- 各字幕の終了時刻は次行の開始時刻の直前。最終行は曲の終わり。
- 現時点では開始時刻タップ記録専用。既存 SRT の読み込み・編集は次段階の候補。

## 開発チェック

Node.js 18+ で `node --test srt-core.test.js` を実行できます。
